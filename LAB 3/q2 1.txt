2. Write a shell script to list all files (only file names) containing the input pattern
(string) in the folder entered by the user.

shell script:

#!/bin/sh

echo "Enter directory"
read dir

echo "Enter pattern: "
read pat

if [ -d "$dir" ]
then 
	ls test | grep "$pat"
else
	echo "Directory doesn't exist"
fi
